# MAUI App (CommunityToolkit) Template

CommunityToolkitが最初から入っているMAUIアプリのテンプレート

## 含まれているもの

- CommunityToolkit.Maui
- CommunityToolkit.Mvvm

## インストール方法

```bash
git clone https://github.com/rkitamu/maui-ctk-template

# install
dotnet new install ./maui-ctk-template/MauiTemplate

# check installed templates
dotnet new list maui-ctk

# create with template
dotnet new maui-ctk -n YourProjectName
```
